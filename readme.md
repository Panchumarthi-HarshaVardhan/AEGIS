# рҹ“– Overview

AEGIS is a next-generation **AI Operating System Guardian** designed to enhance digital safety and productivity.

Unlike conventional antivirus software or AI assistants that react only when asked, AEGIS continuously monitors contextual operating-system events and assists users in making safer decisions.

The system combines Artificial Intelligence, Desktop Automation, Browser Security, and Local Threat Analysis into a unified desktop application.

---

# Problem Statement

Every day users face numerous digital threats:

-  Phishing websites
-  Credential theft
-  Malicious downloads
-  Fake news
-  Deepfake media
-  Online scams
-  Unsafe automation

Traditional solutions solve these problems individually.

**AEGIS brings everything together into one intelligent operating-system companion.**

# вҡҷ Installation

## Clone Repository

```bash
git clone https://github.com/Panchumarthi-HarshaVardhan/AEGIS.git

cd AEGIS
```

## Install Dependencies

```bash
npm install
```

Backend

```bash
cd backend

```

Start Backend

```bash
python main.py
```

Run Electron

```bash
npm run dev
```
